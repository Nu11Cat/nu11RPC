简介

- 轻量级 RPC 框架整体设计与实现，涵盖通信、协议、注册、负载均衡等完整链路能力。

- 基于 Netty 构建 NIO 长连接通信，替代传统 HTTP 通道，实现更高吞吐与更低延迟的高性能 RPC 网络传输层。
- 通过自定义 Invocation 调用结构并编写 Netty Encoder/Decoder，实现自定义 RPC 协议，使客户端与服务端能直接交换方法调用信息。
- 实现包含随机、轮询、加权、加权轮询、一致性哈希在内的多种负载均衡策略，支持可插拔式扩展，提升服务节点调度灵活性与可用性。

----

技术栈

- **语言与框架**：Java 17、Netty、Tomcat
- **RPC 核心实现**：自定义 Invocation 调用结构、JSON/对象序列化、动态代理
- **网络通信**：Netty
- **负载均衡策略**：随机、轮询、加权、加权轮询、一致性哈希
- **服务注册与发现**：Nacos 模拟注册中心与本地注册实现
