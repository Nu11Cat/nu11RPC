- **Bootstrap**
- **common**，提供基础数据结构
  - Invocation：封装调用信息
  - URL：封装服务地址
- **loadbalance**，负载均衡
  - LoadBalance接口
  - 负载均衡算法实现
- **protocol**，网络通信、序列化、反序列化
  - DispatcherServlet：Servlet 入口，调用 HttpServerHandler
  - HttpClient：负责网络调用、发给Provider、返回结果
  - HttpServer：启动 HTTP 服务，接收请求
  - HttpServerHandler：接收请求、解析 Invocation、反射调用本地实现、返回结果
- **proxy**
  - ProxyFactory：生成动态代理对象，拦截方法调用
  - RetryConfig：实现重试策略
- **register**
  - LocalRegister：本地注册接口名和实现类
  - MapRemoteRegister：
    - 本地存储版本：把接口和 URL 写入文件（模拟注册中心）
    - Nacos 版本：实际注册中心注册、发现服务
