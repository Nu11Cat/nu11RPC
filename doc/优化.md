#### 如何使用Netty的

1，使用Netty构建NIO长连接通信。

- 即替换了原来的：HttpServer + HttpClient + Servlet（短连接、阻塞、吞吐差）改成： NettyServer + NettyClient（NIO 长连接）

2，自定义 RPC 编解码协议

3，使用 Netty 的 I/O 线程模型提升吞吐。

- Reactor 多线程模型；零拷贝等；

#### 自定义 RPC 协议如何实现的？

1，定义了RPC 请求结构， RPC 消息体：Invocation

2，定义了编码规则：RpcEncoder

3，定义了解码规则：RpcDecoder

4，定义了处理逻辑：NettyServerHandler

#### 可插拔式扩展

定义了一个统一的策略接口

因为负载均衡算法通过统一接口抽象，各策略独立实现并可在运行期自由替换，新增策略无需修改现有调用逻辑，天然具备可插拔能力。
